TB project
===================

Read the `Documentation <https://camptocamp.github.io/c2cgeoportal/2.7/>`_

Checkout
--------

.. code::

   git clone https://github.com/henriquerolao/tb-geoportal.git
   cd tb-geoportal

Build
-----

.. code::

  sudo python3 build
  sudo docker-compose down

Run
---

.. code::

   docker-compose up -d

.. Feel free to add project-specific things.
